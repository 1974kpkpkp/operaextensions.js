PopupProcess

Methods

ondisconnect

void ondisconnect ()

Parameters:
eventListener - The function that is executed when an injected script or popup's environment is shut down, and can no longer be communicated with.
This event fires when the BackgroundProcess's environment is shut down, and can no longer be communicated with. The source is a messagePort to the disconnecting environment, used only for comparative purposes. The port itself may be closed.

onmessage

void onmessage ()

Parameters:
eventListener - The function that is executed when a message is received from an injected script or the popup.
This event fired when a message is received from the BackgroundProcess. The source is a messagePort to the connecting environment.

postMessage

void postMessage (<DOMString> data, <array> ports)

Parameters:
data - Data to be posted.
ports - An array of ports to be passed along to the BackgroundProcess.
This method is used to post a message to the BackgroundProcess of the extension.
