InjectedScriptScope

Methods

addEventListener

void addEventListener (<DOMString> type, <EventListener> listener, UseCapture)

Parameters:
type - The type of event to listen to. Allowed values are: connect, disconnect and message.
listener - The function that will be called.
useCapture - If true, the event listener is only added for the capture phase and target.
This method is used to listen for events being dispatched by the background process.

ondisconnect

void ondisconnect (<EventListener> eventListener)

Parameters:
eventListener - Function that will listen for disconnection between the extension and page.
This event is fired when the extension is disconnected from the current page (e.g., when the extension is disabled by the user). 
opera.extension.ondisconnect = function(e){
	alert("The extension has been disabled or uninstalled"); 
}

onmessage

void onmessage (<EventListener> eventListener)

Parameters:
eventListener - Function that will listen for messages from the Extension.
This event is used to listen for messages being sent from the background process to the injected script script. Messages from the background process can only be sent as strings and you can access them through the a HTML5 MessageEvent. 
opera.extensions.onmessage = function(e){
	var parsedData = JSON.parse(e.data); 
	
}

postMessage

void postMessage (<DOMString> data, <MessagePortArray> ports)

Parameters:
data - The data to send to the extension.
ports - Optional An array of communication ports as defined in HTML5 Web Messaging.
This method allows for data to be posted directly to the background process.

removeEventListener

void removeEventListener (<DOMString> type, <EventListener> listener, UseCapture)

Parameters:
type - The type of event. Allowed values are: connect, disconnect and message.
listener - The object that the listener was attached to.
useCapture - Which phase the listener was assigned to.
This method removes a listener from receiving an event.
