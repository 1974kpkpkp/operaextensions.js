Popup

Attributes

height

Number height
The height of the popup window. By default, the height is 300px

href

DOMString href
A URL reference to the page to pop up. This page will pop up when the user clicks on an ExtensionUIItem. Editing the href attribute while the popup is open will close the popup.

width

Number width
The width of the popup window. By default, the width is 300px
