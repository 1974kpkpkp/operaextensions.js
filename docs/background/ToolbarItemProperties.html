ToolbarItemProperties

Attributes

badge

Badge badge
This property represents the Badge for the ToolbarItem. 
props.badge = {
  backgroundColor: '#ff0000',
  color: '#ffffff',
  display: 'block',
  textContent: '42'
};

disabled

Boolean disabled
This property indicates that the ToolbarItem needs to be disabled when created. If you don't include it, the default value is false (meaning the created item will be enabled).

icon

DOMString icon
A relative link to a data URI, an image located inside the package, or it can point to an external URL.

popup

Popup popup
This property represents the Popup for the ToolbarItem. 
props.popup = {
  href: 'popup.html',
  width: 200,
  height: 100
};

title

DOMString title
This property represents the title of the item, which is exposed to the user (e.g., as a tooltip when hovering over the item with a mouse).

Methods

onclick

void onclick ()

Parameters:
eventListener - The function that is executed when the ToolbarItem is clicked.
This event fires when the ToolbarItem is clicked. 
var props = {
  disabled: false,
  title: "My Extension",
  icon: "icon.png",
  onclick: function(event) {
    window.opera.postError('Button clicked.');
  }	
};
