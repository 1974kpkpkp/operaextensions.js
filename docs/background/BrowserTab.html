BrowserTab

Attributes

browserWindow

BrowserWindow browserWindow
This is a reference to the BrowserWindow this Tab is contained in.

faviconUrl

DOMString faviconUrl
This attribute contains the URL to the favicon for this tab. This attribute is read only.

focused

boolean focused
This attribute is true when Opera and this tab are in focus. This attribute is read only.

title

DOMString title
This attribute contains the title of the current document in this tab. This attribute is read only.

url

DOMString url
This attribute contains the current URL of this tab. This attribute is read only.

Methods

close

void close ()

This method closes the tab.

update

void update ( <BrowserTabProperties> props )

Parameters:
props - The new properties for the tab. Note: In the current release, only the url property can be updated.
This method changes this tab's properties to the BrowserTabProperties provided.
