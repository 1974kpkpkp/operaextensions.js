Methods

addEventListener

void addEventListener ()

Parameters:
type - Type of event. Allowed values are: focus, close, create and blur.
eventListener - The function to be executed when the event occurs.
useCapture - Boolean, keep false for now. Note: this value currently has no purpose.
This method is used to listen for events being dispatched.

Parameters:

BrowserTab

getFocused

BrowserTab getFocused ()

The property returns the currently focused BrowserTab.
Returns:
{BrowserTab} The currently focused BrowserTab. If Opera itself is not currently in focus or the currently focused tab is not accessible by the background process, this will return null. For a tab to be accessible by the background process, it must have an injected script running in it.

removeEventListener

void removeEventListener ()

Parameters:
type - Type of event. Allowed values are: focus, close, create and blur.
eventListener - The function to be removed.
useCapture - Boolean, keep false for now. Note: this value currently has no purpose.
This method removes a listener from receiving an event.
