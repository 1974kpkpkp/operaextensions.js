Attributes

onconnect

null onconnect
This event listener is invoked when an injected script, popup or options environment is created that can be communicated with. The event's source is a messagePort to the connecting environment. 
opera.extension.onconnect =  function(event)   {
  window.opera.postError('Connected. Event source: ' + event.source);
}

ondisconnect

null ondisconnect
This event listener is invoked when an injected script, popup or options environment is destroyed and can no longer be communicated with. The source is a messagePort to the disconnecting environment, used only for comparative purposes. The port itself may be closed. 
opera.extension.ondisconnect =  function(event)   {
  window.opera.postError('Disconnected. Event source: ' + event.source);
}

onmessage

null onmessage
This event listener is invoked when a message is received from injected script, popup or options page. The source is a messagePort to the connecting environment. 
opera.extension.onmessage =  function(event)  {
  window.opera.postError('Message received. Event source: ' + event.source);
}

tabs

Tabs tabs
This attribute contains the Tabs associated with this extension. Only those tabs which have an injected script from this extension applied to them are available.

windows

BrowserWindows windows
This attribute contains the BrowserWindows associated with this extension. Only those windows which contain tabs accessible by this extension are available.

Methods

addEventListener

void addEventListener (<DOMString> type, eventListerner, <boolean> useCapture)

Parameters:
type - Type of event. Allowed values are: message, disconnect, and connect.
useCapture - Boolean, keep false for now. Note: this value currently has no purpose.
eventListener - The function to be executed when the event occurs.
This method is used to listen for events being dispatched. For the Background Process, this inlcudes "connect", "message", "disconnect". 
opera.extension.addEventListener('connect', function(event) {
  window.opera.postError('Connected. Event source: ' + event.source);
}, false);

opera.extension.addEventListener('message', function(event) {
  window.opera.postError('Message received. Event source: ' + event.source);
  opera.extension.broadcastMessage('Message sent to injected script');
}, false);

opera.extension.addEventListener('disconnect', function(event) {
  window.opera.postError('Disconnected. Event source: ' + event.source);
}, false);
