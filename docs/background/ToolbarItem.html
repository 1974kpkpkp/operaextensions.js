ToolbarItem

Attributes

badge

Badge badge
This read-only property represents the Badge for the ToolbarItem.

disabled

Boolean disabled
This read-only property indicates if the ToolbarItem is disabled. By default, it is set to false (meaning the item is enabled).

icon

DOMString icon
This read-only property represents the icon for the ToolbarItem. It can reference the relative path to a data URI, an image located inside the package, or it can point to an external URL.

popup

Popup popup
This read-only property represents the Popup for the ToolbarItem.

title

DOMString title
This read-only property represents the title of the item, which is exposed to the user (e.g., as a tooltip when hovering over the item with a mouse).

Methods

addEventListener

void addEventListener ()

Parameters:
type - Type of event. Allowed values are: click and remove.
eventListener - The function that is executed when the event occurs.
useCapture - Boolean: should be kept as false for now. Note: this value currently has no purpose.
This method is used to listen for events being dispatched. 
button = opera.contexts.toolbar.createItem(props); // create button
opera.contexts.toolbar.addItem(button); // add button to the browser
button.addEventListener("click", doSomething, false); // add event listener

onclick

void onclick ()

Parameters:
eventListener - The function that is executed when the item in the ToolbarItem is clicked.
This event is fired when the item in the ToolbarItem is clicked. 
var button = opera.contexts.toolbar.createItem(props);
button.addEventListener('click', function(event) {
  window.opera.postError('Button clicked.');
}, false);
opera.contexts.toolbar.addItem(button);

onremove

void onremove ()

Parameters:
eventListener - The function that is executed when the item in the ToolbarItem is removed.
This event is fired when the item in the ToolbarItem is removed. 
var button = opera.contexts.toolbar.createItem(props);
button.addEventListener('remove', function(event) {
  window.opera.postError('Button removed.');
}, false);
opera.contexts.toolbar.addItem(button);

removeEventListener

void removeEventListener ()

Parameters:
type - Type of event. Allowed values are: click and remove.
eventListener - The function to be removed.
useCapture - Boolean: should be kept as false for now. Note: this value currently has no purpose.
This method removes a listener from receiving an event. 
button = opera.contexts.toolbar.createItem(props); // create button
opera.contexts.toolbar.addItem(button); // add button to the browser
button.addEventListener("click", doSomething, false); // add event listener
// Your code goes here
button.removeEventListener("click", doSomething, false); // remove event listener
