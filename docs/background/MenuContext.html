MenuContext

Attributes

length

Number length
This attribute contains the number of UI items held by the MenuContext.

Methods

addItem

void addItem (<MenuItem> item, <MenuItem> before)

Parameters:
item - The MenuItem to be added.
before - The MenuItem to add the first argument before in the context menu (optional).
This method adds a MenuItem to the in-page right click context menus in the browser. To create a MenuItem see the createItem method. 
var item = opera.contexts.menu.createItem({
  title: 'Translate!',
  contexts: ['selection', 'editable']
});
opera.contexts.menu.addItem( item ); // adds it to the in-page right-click context menu

createItem

MenuItem createItem (<MenuItemProperties> props)

Parameters:
props - The MenuItemProperties to use when creating the MenuItem.
This method creates a MenuItem from a given property object. 
var button = opera.contexts.menu.createItem({
  title: 'Translate!',
  contexts: ['selection', 'editable']
});
Returns:
The MenuItem, which at this point is not yet associated with the MenuContext.

item

MenuItem item (<Number> index)

Parameters:
index - The ordinal number of the MenuItem to be retrieved.
This method retrieves a MenuItem by index.
Returns:
{MenuItem} The MenuItem at the requested location.

removeItem

void removeItem (<MenuItem> item)

Parameters:
item - The MenuItem to be removed.
This method removes a given MenuItem from the in-page right-click context menu 
var item = opera.contexts.menu.createItem({
  title: 'Translate!',
  contexts: ['selection', 'editable']
});
opera.contexts.menu.addItem( item ); 
opera.contexts.menu.removeItem( item ); // remove the menu item
