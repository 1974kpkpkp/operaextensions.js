ToolbarContext

Attributes

length

Number length
This attribute contains the number of UI items held by the ToolbarContext. Note: the ToolbarContext is currently restricted to a maximum of one UI item.

Methods

addItem

void addItem (<ToolbarItem> item)

Parameters:
item - The ToolbarItem to be added.
This method adds a ToolbarItem to the toolbar in the browser. To create a ToolbarItem see the createUIITem method. 
var button = opera.contexts.toolbar.createItem({ // creates a button
	title:'Weather Extension',
	icon:'yr.png'
});
opera.contexts.toolbar.addItem( button ); // adds it to the browser UI

createItem

ToolbarItem createItem (<ToolbarItemProperties> props)

Parameters:
props - The ToolbarItemProperties to use when creating the ToolbarItem.
This method creates a ToolbarItem from a given property object. Note: the item can only be used within the ToolbarContext that it is created in. 
var button = opera.contexts.toolbar.createItem({ // create a button
	title:'Weather Extension',
	icon:'yr.png'
});
Returns:
The ToolbarItem, which at this point is not yet associated with the ToolbarContext.

item

ToolbarItem item (<Number> index)

Parameters:
index - The ordinal number of the ToolbarItem to be retrieved.
This method retrieves a ToolbarItem by index.
Returns:
{ToolbarItem} The ToolbarItem at the requested location.

removeItem

void removeItem (<ToolbarItem> item)

Parameters:
item - The ToolbarItem to be removed.
This method removes a given ToolbarItem from the toolbar in the browser. 
var button = opera.contexts.toolbar.createItem({ 
	title:'Weather Extension',
	icon:'yr.png'
});
opera.contexts.toolbar.addItem( button ); 
opera.contexts.toolbar.removeItem( button ); // remove the button
